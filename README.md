# Proyecto de Inteligencia Artificial

Este repositorio contendrá el código de un proyecto de inteligencia artificial

## Contenidos

- `RL_sac/`: notebooks de entrenamiento y test, modelos entrenados y datos del entrenamiento.
    - `modelo_test`: cuaderno para poner a prueba los modelos entrenados y visualizar metricas del entrenamiento
    - `modelo_portable`: cuaderno del entrenamiento y almacenado de modelos. Funciona en collab y de manera local. Almacena y comprueba los modelos entrenados
    - `models/sac_v3/`: modelos entrenados
    - `logs_SAC_v3/`: datos del entrenamiento para leer usando tensorboard
- `requirements.txt` — Dependencias necesarias del entorno.
- `.gitignore`
